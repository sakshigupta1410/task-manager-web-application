# Task Manager Web Application

## 📌 Project Overview

Task Manager is a full-stack web application developed using AngularJS, Node.js, and Express.js. It helps users organize and manage daily tasks efficiently by providing features such as task creation, editing, deletion, filtering, sorting, priority management, subtasks, and dark mode support.

---

## 🚀 Features

* Add new tasks with:

  * Task Name
  * Due Date
  * Category
  * Description
  * Priority Level

* Edit existing tasks

* Delete tasks

* Mark tasks as completed/incomplete

* Search tasks instantly

* Filter tasks:

  * All Tasks
  * Completed Tasks
  * Incomplete Tasks

* Sort tasks by:

  * Due Date
  * Priority
  * Category

* Manage subtasks

* Dark Mode / Light Mode Toggle

* Responsive and user-friendly interface

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/sakshigupta1410/task-manager-web-application
```

### 2. Open Project Folder

```bash
cd task-manager-web-application
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run the Application

```bash
npm start
```

### 5. Open Browser

```text
http://localhost:3000
```

## 📡 API Endpoints

### Get All Tasks

```http
GET /api/tasks
```

### Add Task

```http
POST /api/tasks
```

### Update Task

```http
PUT /api/tasks/:id
```

### Delete Task

```http
DELETE /api/tasks/:id
```
